TOAST 서비스

기업의 운영과 서비스 개발을 위해 필요한 모든 것을 제공합니다.

다양한 서비스를 체험해보세요.

Compute

  • Instance

    원하는 즉시 생성할 수 있는 가상 서버입니다.

  • Image

    OS와 함께 기본적으로 설치 제공하는 애플리케이션 템플릿입니다.

  • Auto Scale

    다양한 이벤트에 따라 인프라 증설이 가능한 서비스입니다.

Network

  • VPC

    가상 사설 네트워크를 제공합니다.

  • Load Balancer

    대규모 트래픽 분산 기능을 제공합니다.

Storage

  • Block Storage

    Instance에 할당하여 사용할 수 있는 블록 레벨 스토리지입니다.

  • NAS (offline)

    여러 개의 인스턴스에서 네트워크를 통해 액세스할 수 있는 스토리지입니다.

  • Object Storage

    REST API 기반의 확장 가능한 대용량 오브젝트 스토리지입니다.

  • Backup

    높은 신뢰성과 안정성을 가진 백업 서비스입니다.

Database

  • RDS for MySQL

    복잡한 설치 및 설정 과정 없이, 클라우드 환경에서 관계형 데이터베이스를 사용할 수 있습니다.

Analytics

  • App Analytics

    Forecasting Engine을 기반으로 신뢰성 있는 예측 데이터를 제공합니다.

  • Log & Crash Search

    클라이언트와 서버의 로그를 수집하여 사용자가 원하는 로그를 검색하고 조회할 수 있습니다.

Game

  • Gamebase

    손쉽고 효율적으로 게임을 개발할 수 있게 돕는 서비스입니다.

  • Leaderboard

    대규모 트래픽에서도 안정적으로 동작하는 실시간 랭킹 서비스입니다.

  • Launching

    모바일 앱을 서비스할 때 필요한 론칭 관리 서비스입니다.

  • Smart Downloader

    게임을 개발할 때 리소스를 효율적으로 다운로드하고, 다운로드 통계 데이터를 수집해서 다양한 정보를 확인할 수 있습니다.

Security

  • AppGuard

    애플리케이션의 코드 조작을 방지할 수 있습니다.

  • Security Check

    보안 취약점 검수를 수행하고 취약점에 대한 대응 방법을 제공하는 서비스입니다.

  • Security Monitoring

    검증된 IDS/SIEM 패턴 및 위협 관리 대응 체계를 기반으로 한 보안 관제 서비스입니다.

  • Mal-URL Detector

    웹 사이트 해킹을 통해 악성코드를 유포하거나 경유 사이트로 악용되는 것을 사전에 탐지하고, 이를 대응할 수 있도록 서비스를 제공합니다.

  • CAPTCHA

    이미지 CAPTCHA, 음성 CAPTCHA를 사용할 수 있도록 API를 제공합니다.

  • OTP

    OTP 인증을 위한 API를 제공합니다.

  • DBSafer

    접근 통제, 권한 통제, SQL 감사 및 로깅 등을 수행하는 서비스입니다.

  • Web Firewall

    고객 클라우드 영역에 웹 방화벽이 구성되며, 독립적인 운영 환경을 제공합니다.

  • Vaccine

    기술력을 인정받은 Trend Micro Deep Security 제품을 통해 악성코드로부터 이용자의 서버를 보호합니다.

Content Delivery

  • CDN

    빠른 속도로 콘텐츠를 배포할 수 있습니다.

  • Image

    이미지 저장, 편집, 전송까지 아우르는 이미지 시스템입니다.

Mobile Service

  • IAP

    모바일 통합 인앱 결제 서비스를 제공합니다.

  • Mobile Device Info

    다양한 디바이스를 모델 코드, 모델명으로 조회할 수 있습니다.

Notification

  • Push

    다양한 전송 방법으로 메시지를 전송하고 그 결과를 조회할 수 있습니다.

  • SMS

    SMS/MMS 발송, 예약 발송, 조회, 템플릿 관리까지 아우르는 SMS 발송 시스템입니다.

  • Email

    메일 발송, 예약 발송, 조회, 템플릿 관리까지 아우르는 메일 발송 시스템입니다.

  • KakaoTalk Bizmessage

    카카오톡 사용자에게 배송, 예약 시간 등의 정보성 메시지를 발송할 수 있는 알림톡과 광고성 메시지를 발송할 수 있는 친구톡을 함께 제공하는 서비스입니다.

Application Service

  • Maps

    웹 지도, 통합 검색, 경로 탐색을 제공하는 API 웹 지도 서비스입니다.

  • ROLE

    역할 기반 접근 제어(role-based access control, RBAC)를 쉽게 사용할 수 있도록 UI 및 REST API를 제공합니다.

  • API Gateway

    API를 생성, 관리, 배포, 모니터링할 수 있습니다.

  • RTCS

    Socket.IO 기반으로 웹과 모바일 등을 사용하는 사용자와 서비스 간 실시간 메시지를 주고받을 수 있습니다.

  • CS & BBS

    고객을 위한 기본 CS 화면(공지 사항, FAQ, 1:1 문의)을 제공하고 CS 콘텐츠 관리 기능을 제공합니다.

Search

  • Cloud Search

    DBMS 검색과 정보 검색의 단점을 보완한 서비스입니다.

  • Autocomplete

    자동 완성 서비스를 합리적인 비용으로 손쉽게 구축할 수 있습니다.

  • Corporation Search

    사업자등록번호에 대하여 과세 유형 및 신규 등록, 휴/폐업 여부를 조회할 수 있습니다.

  • Address Search

    도로명주소, 지번주소 등을 편리하게 검색할 수 있는 API를 제공합니다.

Dev Tool

  • Deploy

    서버 애플리케이션과 클라이언트(스마트폰) 설치 애플리케이션의 배포 서비스를 제공합니다.

Management

  • Managed

    전문 지식을 겸비한 전담 엔지니어가 구축부터 운영, 백업까지 서비스를 안정적으로 관리합니다.

Dooray!

  • Project

    자동 분류 규칙을 이용해 쉽게 관리할 수 있으며 자동으로 업무를 등록할 수도 있습니다.

  • Messenger

    업무의 속도를 높이는 실시간 커뮤니케이션 서비스입니다.

  • Mail

    업무에 최적화된 똑똑한 메일 서비스입니다.

  • Calendar

    업무 일정을 관리할 수 있는 캘린더 서비스입니다.

  • Drive

    내 드라이브뿐 아니라, 프로젝트 드라이브를 제공해 멤버들이 함께 파일을 저장하고 관리할 수 있습니다.

  • Contacts

    개인/공용 주소록 서비스를 제공합니다.

ERP

  • Communication Board

    양방향 소통을 지원하는 그룹 기반의 커뮤니케이션 서비스입니다.

  • Workflow

    업무 환경의 효율성을 높이는 ERP 기반의 전자 결재 서비스입니다.

  • Human Resources

    효과적인 인사 관리를 위한 쉽고 편리한 인사 시스템입니다.

  • Finance

    회계 업무의 솔루션을 담은 재무 회계 시스템입니다.

Bill

  • Bill (e-Tax)

    쉽고 편리하게 세금계산서를 발행할 수 있습니다.

Contact Center

  • Omni Contact

    콜센터 통합 상담 솔루션입니다.

  • Mobile Contact

    고객 문의에 보다 체계적, 탄력적, 전문적으로 대응할 수 있습니다.

  • Online Contact

    유형별 분류를 통해 고객 센터 웹 페이지에 다양한 형태로 정보를 제공하고 관리하는 시스템입니다.

IDC

  • TCC

    자체 기술력으로 경기도 판교에 설계, 구축한 도심형 인터넷 데이터 센터입니다.
