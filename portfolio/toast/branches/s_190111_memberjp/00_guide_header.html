  • 기본 : 투명 배경&상단 고정 안함
  • 상단 고정 : header에 is_fixed 클래스 추가
  • 불투명배경(흰색) 적용 : header에 is_hover 클래스 추가

다른 사이트 공용 헤더 적용시 필요한 파일

  • CSS 파일 : gnb.css
  • 이미지 파일 : sp_gnb.png

하위메뉴 추가 가이드

  • 1. 서브메뉴가 필요한 헤더에 클래스 추가
  • 2. 서브메뉴가 필요한 메뉴(link_txt)다음에 drop_menu_wrap 영역 추가 한 후 drop_menu_wrap에 클래스 추가
  • 3. 서브메뉴 위치를 추가한 클래스의 padding-left 값으로 조정
