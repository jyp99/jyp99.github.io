추가 적용 JS컴포넌트

추가 필요한 JS컴포넌트 입니다.
다운받아 제공되는 css도 함께 적용하시면 되고, css 재지정은 샵웹 마크업에서 이미 선언되어 있습니다.

  • TUI Datepicker (Latest) https://github.com/nhnent/tui.date-picker
  • TUI Pagination (Latest) https://github.com/nhnent/tui.pagination

css 로드 시 스타일 선언 순서로 인해 다음 예시와 같이 solution.css가 가장 마지막에 위치해야 합니다. 


	<link rel="stylesheet" href="js/bower_components/tui-date-picker/dist/tui-date-picker.css">
	<link rel="stylesheet" href="js/bower_components/tui-pagination/dist/tui-pagination.css">
	<link rel="stylesheet" href="css/solution.css">
