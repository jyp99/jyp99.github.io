1. ประเด็นของข้อมูลที่เก็บและวิธีเก็บข้อมูล

  1. ก. ประเด็นของข้อมูลที่เก็บ
    เพื่อสมัครสมาชิก PAYCO, การให้คำปรึกษาแก่ลูกค้า, รวมทั้งการให้บริการ ต่างๆ
    ทางบริษัทเก็บข้อมูลส่วนบุคคลโดยมีประเด็น ได้แก่
    1. - E-mail หรือ หมายเลขโทรศัพท์มือถือ (PAYCO ID) รหัสผ่าน
      นอกจากนี้ในขั้นตอนต่างๆ ของการใช้ บริการ มีการเก็บข้อมูลเพิ่มเติม ดังนี้
    2. - IP Address, cookie, วันเวลาเข้าถึง, ประวัติการใช้บริการตามปกติ, ประวัติการใช้บริการที่ผิดปกติ,
      ข้อมูลอุปกรณ์โทรศัพท์มือถือ (รุ่นอุปกรณ์, เครือข่ายที่ใช้, hardware ID,
      ข้อมูลสถิติพื้นฐานสำหรับการใช้บริการ), ข้อมูล application ที่ติดตั้งอยู่ในอุปกรณ์, ข้อมูลการจ่ายเงิน
  2. ข. วิธีเก็บข้อมูลส่วนบุคคล
    บริษัทจะเก็บข้อมูลส่วนบุคคลโดยใช้วิธีดังต่อไปนี้
    1. - หน้าแรก (homepage), อุปกรณ์มือถือ, เอกสารที่เป็นลายลักษณ์อักษร, แฟกซ์(fax), โทรศัพท์,
      กระดานประกาศ, อีเมลล์(e-mail), การชิงโครผ่านอีเวนต์(Event), การขอส่งทางไปรษณีย์
    2. - บริษัทที่สัมพันธ์ร่วม
    3. - เครื่องมือที่ช่วยเพิ่มพูนข้อมูลเพื่อใช้บริการ
  3. นอกจากนี้บริษัทยังสามารถเก็บข้อมูลเพิ่มเติมได้
    แต่สำหรับกรณีนี้บริษัทจะขออนุญาตจากผู้ใช้ก่อนล่วงหน้า

    4. 2. การเก็บข้อมูลส่วนบุคคลและวัตถุประสงค์การใช้

    1. บริษัทเก็บข้อมูลส่วนบุคคลและใช้โดยมีวัตถุประสงค์ดังต่อไปนี้
      1. ก. การปฏิบัติตามหนังสือสัญญาเรื่องการให้บริการและการคิดคำนวณค่าของการให้บริการ
        การให้ คอนเทนช์, การส่งสินค้า หรือส่งใบแจ้งหนี้, การทำตรวจสอบหรือยืนยันผู้ใช้,
        การซื้อหรือการจ่ายเงิน, การตามหาเพื่อชำระเงิน
      2. ข. การบริหารสมาชิก
        การให้บริการเฉพาะแก่สมาชิก, การตรวจสอบแต่ละคนได้,
        การป้องกัน มิให้สมาชิกที่ประพฤติไม่พึงประสงค์ (สมาชิกที่ฝ่าฝืนข้อกฎบังคับ
        จึงได้รับการถูกจำกัดการใช้หรือถูกถอนออกจากสมาชิก)
        ใช้ บริการในทางที่ไม่ดีและในทางที่ไม่ได้รับอนุญาต, การตรวจสอบความสมัครใจในการเป็นสมาชิก,
        การจำกัดจำนวนการสมัคร สมาชิก
        การยินยอมจากผู้ปกครองหากจะเก็บข้อมูลสมาชิกที่ยังไม่บรรลุนิติภาวะ,
        การตรวจสอบหรือการยืนยันผู้ปกครองจริง, การเก็บ ประวัติไว้เพื่อไกล่เกลี่ยความขัดแย้งที่อาจเกิดขึ้น,
        การแก้ไขความไม่พอใจ, การแจ้งประกาศข้อมูลต่างๆ
      3. ค. การนำมาใช้เพื่อการพัฒนาบริการใหม่ การโฆษณาและการตลาด
        การพัฒนาบริการใหม่และการ ให้บริการให้สมกับความต้องการ,
        การให้บริการและการโฆษณาตามลักษณะเชิงประชากรศาสตร์,
        การตรวจสอบหรือการยืนยันความ มีผลของบริการ, การให้ข้อมูลเชิงพาณิชย์และอีเวนต์
        รวมทั้งให้โอกาสมีส่วนร่วม, การตรวจสอบความบ่อยครั้งในการเช้าถึงหรือการใช้ บริการ,
        การสรุปผลเชิงสถิติของการใช้บริการของสมาชิกล

    3. ระยะเวลาการเก็บและใช้ข้อมูลส่วนบุคคล

    1. โดยหลักการ
      ข้อมูลส่วนบุคคลของผู้ใช้ถูกลบทิ้งไปเมื่อนำมาใช้เพื่อบรรลุวัตถุประสงค์การเก็บและการใช้ข้อมูล ส่วนบุคค
      ล แต่เพื่อให้ความช่วยเหลือแก่ผู้ที่ลาออกจากสมาชิกโดยไม่ได้ตั้งใจ
      บริษัทก็เก็บข้อมูลส่วนบุคคลของผู้ที่ขอลาออกจาก สมาชิกเป็นระยะเวลา 7 วัน
      สำหรับกรณีดังต่อไปนี้เป็นกรณีพิเศษที่บริษัทเก็บไว้โดยมีเหตุผลและระยะเวลาตามกำหนด
      จะไม่นำไปใช้ ในวัตถุประสงค์อื่นๆ อย่างเด็ดขาด
      1. ก. กรณีตามนโยบายของบริษัท
      2. ข. กรณีตามข้อกฎหมายที่เกี่ยวข้อง
        ประมวลกฎหมาที่เกี่ยวข้อง เช่น กฎหมายว่าด้วยการพาณิชย์ และกฎหมายว่าด้วยการคุ้มครองผู้บริโภค
        เป็นต้น ระบุให้บริษัทเก็บข้อมูลส่วนบุคคลตามระยะเวลาและวัตถุประสงค์ที่กำหนดไว้
        เหตุผลและระยะเวลา ดังนี้
        - ข้อมูลการเยี่ยมเว็บไซต์
        เหตุผลที่เก็บ : กฎหมายว่าด้วยการรักษาความลับทาง โทรคมนาคม
        ระยะเวลาที่กำหนด : 3 เดือน
        - ข้อมูลเกี่ยวกับการจัดการแก้ไขความไม่พอใจของผู้บริโภคหรือความขัด แย้ง
        เหตุผลที่เก็บ : กฎหมายว่าด้วยการคุ้มครองผู้บริโภคในการซื้อขายทางอิเล็กทรอนิกส์
        ระยะเวลาที่เก็บ : 3ปี
        - ข้อมูลเกี่ยว กับการลงนามสัญญา หรือยกเลิกการลงนามข้างท้าย
        เหตุผลที่เก็บ : กฎหมายว่าด้วยการคุ้มครองผู้บริโภคในการซื้อขายทาง อิเล็กทรอนิกส์
        ระยะเวลาที่เก็บ : 5ปี
      3. - ข้อมูลเกี่ยวกับการจ่ายค่าบริการและการขนส่งสินค้า
        เหตุผลที่เก็บ : กฎหมายว่าด้วยการคุ้มครองผู้ บริโภคในการซื้อขายทางอิเล็กทรอนิกส์
        ระยะเวลาที่เก็บ : 5ปี
