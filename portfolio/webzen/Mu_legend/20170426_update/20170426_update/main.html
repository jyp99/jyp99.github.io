상위 등급의 기술문장 업데이트

기술문장 큐브에서 상위 등급의 기술문장을 획득할 수 있습니다.

참고사항

  • 상위 등급의 기술문장이 추가 되었으며, 최대 희귀 등급까지의 기술문장을 획득할 수 있습니다.
  • 상위 등급의 기술문장을 사용함으로써 기존의 일반 기술문장보다 더욱 큰 효과를 발휘하게 됩니다.
  • 상위 등급의 기술문장은 기술문장 큐브에서 획득 가능합니다.
  • 기술문장 큐브는 블러드 캐슬에서 획득 가능합니다.
  • 기술문장 큐브 사용 시 사용한 클래스가 사용할 수 있는 문장이 자동으로 획득됩니다.

3:3 협동전 월드매칭 업데이트

이제 다른 월드의 수호자 분들과 3:3 협동전 매칭이 가능해 집니다.

참고사항

  • 기존 서버 내 매칭에서 월드 매칭으로 매칭 범위가 확대됩니다.
  • 매칭 시 하나의 클래스로만 몰리지 않도록 매칭 룰이 개편되어 다양한 클래스와 파티를 맺을 수 있게 됩니다.
